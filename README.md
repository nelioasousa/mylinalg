# Description
Final project for the Computational Linear Algebra course.

# Implementations
## Part 1
1. Gauss elimination;
2. Gauss-Jordan;
3. LU and PLU decomposition for solving linear equations;
4. RREF;
5. Cholesky decomposition;
6. Gram-Schmidt orthogonalization;
7. Least squares solution for linear equations;

## Part 2
8. Power iteration;
9. Householder's method;
10. QR decomposition;
11. QR algorithm for eigenvalues and eigenvectors;
12. SVD decomposition;

## Part 3
13. Successive Over-Relaxation (SOR);
14. Conjugate gradient method.
